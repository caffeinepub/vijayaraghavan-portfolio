- Apply glassmorphism design system with frosted glass cards, neubrutalism accents, vibrant gradients, and full responsive layout

**User-visible outcome:** A fully animated, ultra-modern portfolio landing page with futuristic aesthetics, smooth scroll animations, interactive 3D effects, glassmorphism design, particle backgrounds, and responsive dark/light mode support showcasing Vijayaraghavan's full-stack development skills.
